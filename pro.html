Frequently asked questions

What happens if the license server is down?

Pro features keep working. Validation is cached for 24 hours and license keys are cryptographically signed, so a server outage never locks paying users out.

Can I use my license on multiple machines?

Yes — each license supports 3 machine activations. If you need more (e.g. a build server), contact support.

What's the refund policy?

If a Pro feature isn't working as described, email within 14 days for a full refund. No questions asked.

Do free features stop working if I don't subscribe?

Never. The free tier is permanent and fully functional. Pro only adds advanced features on top.
